Lestia
------
[![Build Status](https://travis-ci.org/pine613/lestia.svg?branch=master)](https://travis-ci.org/pine613/lestia)

Lestia is a library of Message-based RPC that can do callback.

## Get started

```
$ npm install lestia --save
```

## Tests

```
$ npm install
$ npm test
```
